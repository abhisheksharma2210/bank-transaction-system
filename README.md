# 💳 Bank Transaction System API

## 🚀 Features
- User Registration & Login (JWT Authentication)
- Account Creation
- Deposit Money
- Withdraw Money
- Fund Transfer between accounts

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT Authentication
